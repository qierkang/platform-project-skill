# 环境变量映射说明

## 根级变量

| 变量 | 说明 | 默认值 |
|---|---|---|
| `SERVER_PORT` | 服务端端口 | `7060` |
| `FRONT_PORT` | 管理后台端口 | `7061` |
| `MOBILE_PORT` | 移动端端口 | `7062` |
| `POSTGRES_PORT` | PostgreSQL 对外端口 | `55432` |
| `REDIS_PORT` | Redis 对外端口 | `56379` |
| `API_BASE_URL` | 前端/移动端访问服务端的地址 | `http://127.0.0.1:7060` |

## 派生建议

- 新项目派生时，优先只改根级 `.env`
- 不建议把端口、数据库名分散写到多个子项目 README 或脚本中

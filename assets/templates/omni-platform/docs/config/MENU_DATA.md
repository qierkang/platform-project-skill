# omni-platform 菜单配置

## 菜单结构

- 当前母版不内置真实业务菜单。
- 后续派生项目建议从以下最小结构开始：

| 菜单编码 | 菜单名称 | 所属端 | 默认状态 | 说明 |
|---|---|---|---|---|
| `dashboard` | 工作台 | Web | 待实现 | 管理端首页入口 |
| `docs` | 文档中心 | Web | 可配置 | 跳转到项目文档入口 |
| `mobile-home` | 移动首页 | H5 | 待实现 | 移动端默认入口 |

## 权限点

- 当前母版不内置权限点。
- 后续派生时，建议按 `模块:资源:动作` 命名。

| 权限点 | 说明 |
|---|---|
| `platform:dashboard:view` | 查看工作台 |
| `platform:docs:view` | 查看文档入口 |

## 执行说明

- 菜单和权限点在真实业务项目中落库或接入权限系统。
- 本文件只作为配置模板和命名参考，不作为可执行 SQL。

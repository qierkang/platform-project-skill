# UI Design Image Prompt

Use this prompt when generating front or mobile UI design drafts with `image_gen`.

## Front UI

```text
手绘风格 UI 设计稿，标题“{项目名} 管理端 UI 设计稿”。展示后台首页、侧边导航、顶部状态区、数据卡片、表格、流程操作区、配置入口。风格干净、结构清晰、中文标签、白底手绘线框、少量蓝绿橙强调色，适合 README 展示。
```

## Mobile UI

```text
手绘风格移动端 UI 设计稿，标题“{项目名} 移动端 UI 设计稿”。展示移动首页、待办卡片、审批流程、消息列表、个人中心、底部导航。白底手绘线框、中文标签、适合 H5 / Mobile Web 项目 README 展示。
```

## Required Save Paths

```text
assets/{project-slug}-front/design/front-ui-design-draft.png
assets/{project-slug}-mobile/design/mobile-ui-design-draft.png
```

# Project Baseline Image Prompt

Use this prompt when generating a platform design baseline image with `image_gen`.

```text
手绘风格平台设计基线图，标题“{项目名} 设计基线”。展示视觉语言、色彩、信息架构、页面布局、组件层、状态反馈、PC 管理端和移动端一致性。白底、中文标签、线框草图、蓝绿橙强调色，适合平台母版 README 展示。
```

## Required Save Path

```text
assets/platform/design/{project-slug}-design-baseline.png
```

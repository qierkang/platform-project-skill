# Platform Project Skill Start Here

This skill creates new platform projects and upgrades existing projects with an AI handoff layer.

## Quick Route

- New project: read `SKILL.md`, then `references/workflow-new-project.md`.
- Existing project: read `SKILL.md`, then `references/workflow-existing-project.md` and `references/non-invasive-upgrade.md`.
- README work: read `references/readme-rules.md`.
- Assets and diagrams: read `references/assets-rules.md`.
- Graphify work: read `references/graphify-rules.md`.

## Core Promise

- New projects should closely reproduce the bundled `omni-platform` mother template.
- Existing projects should keep their original structure and receive only AI collaboration upgrades.
